# Exam Timer ⏱️

A simple and stylish **Exam Timer** web application built with **HTML, CSS, and JavaScript**.  
This project allows students or instructors to set a countdown timer for exams, track the remaining time, and reset when needed.

---

## Live Demo
Live on: [GitHub Pages](https://guitareren.github.io/bilgitimer/)
Neon Theme is Live on: [GitHub Pages](https://guitareren.github.io/bilgitimer/neon.html)
## Features
- ⏳ Set custom exam duration (hours, minutes, seconds).
- 🎯 Start, pause, and resume the timer.
- 🔄 Reset the timer back to `00:00:00`.
- ⚡ Animated and responsive UI with modern CSS effects.
- 🖥️ Works directly in the browser, no installation required.

## Usage

- Open the project folder and run the **index.html** file in your browser.  
- Enter the exam duration using the input fields:  
  - **Hours**  
  - **Minutes**  
  - **Seconds**  
- Click **Start Exam** to begin the countdown.  
- The button changes to **Pause Exam** while running.  
- If paused, it changes to **Resume Exam**.  
- When the timer reaches zero, the display shows **TIME UP!**.  
- Use the **Reset** button to clear the timer and start again.
